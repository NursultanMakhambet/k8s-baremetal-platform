# Platform namespaces

| Namespace    | Purpose |
|-------------|---------|
| **monitoring** | Grafana, VictoriaMetrics, Alertmanager, exporters. |
| **argocd**     | Argo CD (GitOps). |
| **vault**      | HashiCorp Vault, if used. |
| **platform**   | cert-manager, ingress, external-secrets, other shared services. |
